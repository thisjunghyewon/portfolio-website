  • Name of the project

    Meet

  • Overview

    The Meet App enables users to explore upcoming events in a particular city through the integration of the Google Calendar API. This React App follows a Test-Driven Development (TDD) approach, seamlessly blending the principles of serverless architecture and progressive web applications.

  • Purpose & Context

    During my web development course at CareerFoundry, I created the Meet project to demonstrate my proficiency in test-driven development, such as building a Progressive Web App (PWA) construction. Implementing data visualization into the UI of a web app was my final destination.

  • Objective

    This project aimed to build a progressive web app that works offline and a serverless backend built by applying test-driven development (TDD) methodology.
